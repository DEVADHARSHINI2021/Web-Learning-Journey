# Day 10 – Currency Converter 💱

## 📌 Project Overview

A simple Currency Converter built using **HTML, CSS, and JavaScript**. This project fetches real-time exchange rates from a public API and converts an amount from one currency to another.

---

## 🚀 Features

* Convert between multiple currencies
* Real-time exchange rates using an API
* Clean and responsive user interface
* Simple and beginner-friendly code
* Uses asynchronous JavaScript (`async/await`)

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* JSON

---

## 📚 Concepts Learned

* Working with APIs
* `fetch()`
* `async` and `await`
* Parsing JSON data
* DOM Manipulation
* Event Listeners
* Reading values from `<select>` elements
* Template Literals
* Basic currency conversion logic

---

## ▶️ How to Run

1. Download or clone the project.
2. Open `index.html` in your web browser.
3. Enter an amount.
4. Choose the source and target currencies.
5. Click **Convert** to see the converted amount.

---

## 🎯 What I Learned

This project helped me understand how JavaScript communicates with external APIs to retrieve live data. I also learned how to work with asynchronous programming using `async` and `await`, parse JSON responses, and dynamically update webpage content based on user input.
