# 📅 Day 14 – Image Slider

## 📖 Project Overview

The **Image Slider** is a simple web application that allows users to browse through a collection of images using **Previous** and **Next** buttons. It demonstrates how JavaScript can dynamically update HTML elements and manage data using arrays.

This project focuses on manipulating the DOM, handling user interactions, and implementing navigation logic with JavaScript.

---

## 🚀 Features

* Display images from a predefined array
* Navigate to the next image
* Navigate to the previous image
* Automatic wrap-around navigation
* Responsive and clean user interface

---

## 🛠️ Technologies Used

* CSS3
* JavaScript (ES6)
